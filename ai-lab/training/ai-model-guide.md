*Standard Operating Procedure for selecting the right AI for the job.*

## 1. DeepSeek (Coder & Chat)
**Best For:** Coding, Debugging, Logic Puzzles.
* **Strengths:** Extremely strong at Python, JSON, and fixing server scripts. Open-source friendly.
* **When to use:**
    * "My Python script has an indentation error."
    * "Write a Linux bash script to backup SQL databases."
* **Cost:** Low/Free options available.

## 2. Claude 3.5 Sonnet (Anthropic)
**Best For:** Writing, Human-like Nuance, Long Context Analysis.
* **Strengths:** Reads large PDF documents or long code files without forgetting the beginning. Very natural tone for emails.
* **When to use:**
    * "Rewrite this angry email to a client to sound professional."
    * "Analyze this 50-page server log and find the anomaly."

## 3. ChatGPT (GPT-4o)
**Best For:** General Knowledge, Quick Answers, Image Generation.
* **Strengths:** The "Jack of all trades." Good for brainstorming and quick fixes.
* **When to use:**
    * "Give me 10 ideas for a marketing post."
    * "Translate this error message from German to English."
